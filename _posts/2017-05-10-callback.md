title: '回调函数'
---
一个通过函数指针调用的函数。如果你把函数的指针（地址）作为参数传递给另一个函数，当这个指针被用为调用它所指向的函数时，我们就说这是回调函数。 
<!--break-->
### 1.eg1
function a(callback) {
    callback();
}

function b() {
    console.log('hello callback');
}

a(b); 
### 2.eg2
function print(){
	console.log('callback');
}
setTimeout(print,1000);
